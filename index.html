Lambda expressions

Java (lambdas) 
OptionalInt sum =
  IntStream.of(1, 2, 3, 4, 5)
    // 1, 3, 5
  .filter(i -> (i % 2) != 0)
  // 2, 4, 6
  .map((i) -> { i++; return i; })
  // 2 + 4 + 6
  .reduce((i, a) -> a + i);
ES6 (arrows) 
let sum =
  Array.of(1, 2, 3, 4, 5)
  // 1, 3, 5
  .filter(i => (i % 2) === 0)
  // 2, 4, 6
  .map((i) => { i++; return i; })
  // 2 + 4 + 6
  .reduce((i, a) => i + a);

Modules

In ES6 file is named module. It may import or export functions, objects or primitives from or to other modules

// example1.js

// example2.js

Component could be stateful
like any Java class :-)

class TogleButton extends React.Component {
  constructor(props) {
    super(props);
    this.state = { pushed: true };
  }

  togle() {
    this.setState((prevState) => ({
      pushed: !prevState.pushed
    }));
  }

  render() {
    return (
      <input
      	type="button"
      	className={this.state.pushed ? 'up' : 'down'}
        onClick={this.togle}
        value="Togle"
      />
    );
  }
}
